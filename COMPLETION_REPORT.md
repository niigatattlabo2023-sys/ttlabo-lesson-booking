## 🚀 使い方（すぐに試せます）

### ステップ1: ブラウザで開く
`index.html` をブラウザで開いてください

### ステップ2: 予約してみる
「予約する」ボタンから試しに予約申請

### ステップ3: 管理画面で承認
1. `admin.html` を開く
2. ログイン（Supabaseで作成したアカウント）
3. 予約を承認
4. **施設予約サポート**が表示されることを確認
5. 電話ボタン/Webボタンをクリック

## 💡 次にやること

### Supabaseの設定（まだの場合）
1. **SQL実行:**
   - Supabase Dashboard → SQL Editor
   - `supabase-schema.sql` の内容をコピペして実行

2. **管理者アカウント作成:**
   - Supabase Dashboard → Authentication → Users
   - 「Add user」でメールアドレス+パスワード設定

3. **施設情報の登録:**
   - 管理画面の「施設管理」タブ
   - 電話番号とURLを登録（施設予約サポート用）

## 🎯 施設予約サポートを使うためのポイント

### 必須設定
施設マスタに以下のいずれかを登録：
- ✅ **電話番号**（例：025-382-3222）
- ✅ **予約URL**（例：https://facility-booking.com）

### 表示条件
以下の条件を満たすと自動表示：
- ✅ 予約ステータスが「承認済み」
- ✅ 施設予約担当が「コーチ」
- ✅ 施設に電話番号またはURLが登録済み
