# 📸 施設予約サポート機能 - ビジュアルガイド

## 🎯 この機能の目的

予約を承認した後、施設に電話やWeb予約をする手間を**大幅に削減**します。

## 🎬 操作の流れ（アニメーション風）

### STEP 1: 承認ボタンをクリック
```
[承認✅] ← クリック
```
↓

### STEP 2: 施設予約サポートが表示される
```
📞 施設予約が必要です
亀田総合体育館

[📞 電話をかける] [🌐 Web予約ページを開く]
```
↓

### STEP 3A: 電話予約の場合
```
[📞 電話をかける] ← タップ
   ↓
📱 スマホの電話アプリが起動
   ↓
025-382-3222 に発信
   ↓
🗣️ 施設に予約完了
   ↓
☑️ チェックボックスにチェック
```

### STEP 3B: Web予約の場合
```
[🌐 Web予約ページを開く] ← クリック
   ↓
🌐 別タブで予約ページが開く
   ↓
💻 予約手続き完了
   ↓
☑️ チェックボックスにチェック
```

### STEP 4: LINEメッセージをコピー
```
[コピーする] ← クリック
   ↓
📋 クリップボードにコピー完了
   ↓
💬 LINEで顧客に送信
```

## 🎨 デザインの特徴

### カラー
- **背景:** グラデーション（黄色 → 青）で目立つ
- **電話ボタン:** 緑色（発信のイメージ）
- **Webボタン:** 青色（リンクのイメージ）

### アイコン
- 📞 電話
- 🌐 Web
- ✅ 完了チェック
- 💬 LINEメッセージ

### レスポンシブ
- **PC:** 横並び2カラム
- **スマホ:** 縦並び1カラム

## 💪 メリット一覧

### ✅ 時間短縮
| 項目 | 今まで | これから |
|------|--------|----------|
| 電話番号を探す | 30秒 | 0秒（自動表示） |
| 電話をかける | 20秒 | 1秒（タップ） |
| URLを探す | 40秒 | 0秒（自動表示） |
| ブラウザで開く | 20秒 | 1秒（クリック） |
| **合計** | **約2分** | **約10秒** 🚀 |

### ✅ ミス削減
- ❌ 電話番号の入力ミス → ✅ 自動で正確
- ❌ URLの打ち間違い → ✅ 自動でリンク
- ❌ 予約忘れ → ✅ チェックボックスで管理

### ✅ 使いやすさ
- 大きなボタン
- わかりやすいアイコン
- スマホでもタップしやすい

## 🔧 カスタマイズポイント

### 施設情報の編集
管理画面の「施設管理」タブで以下を設定：

```
施設名: 亀田総合体育館
住所: 新潟市江南区茅野山3-1-14
電話番号: 025-382-3222 ← これを登録
予約URL: https://example.com/booking ← これを登録
予約方法: 両方（電話とWeb）
```

**電話番号またはURLを登録すると、施設予約サポートが使えます！**
