### 2. 施設予約サポート機能の強化 🚀

#### 実装内容
管理画面で予約を承認すると、**施設予約が必要な場合に自動的にサポート機能が表示**されます。

#### 表示条件
- 施設予約担当が「コーチ」
- 予約ステータスが「承認済み」
- 施設情報（電話番号またはURL）が登録済み

#### できること

**📞 電話予約の場合:**
```
┌────────────────────────┐
│ [📞 電話をかける]       │
│  025-XXX-XXXX          │
└────────────────────────┘
```
- ボタンをタップするとスマホの電話アプリが起動
- 電話番号を手入力する必要なし
- ワンタップで施設に電話できる

**🌐 Web予約の場合:**
```
┌────────────────────────┐
│ [🌐 Web予約ページを開く]│
└────────────────────────┘
```
- ボタンをクリックすると施設の予約ページが別タブで開く
- URLをコピペする手間なし
- そのまま施設予約手続きができる

**✅ 予約完了チェック:**
```
┌────────────────────────────────┐
│ ☑️ 施設予約を完了しました      │
└────────────────────────────────┘
```
- 施設予約が終わったらチェック
- 予約漏れ防止に便利

## 📱 実際の使い方の流れ

### ステップ1: お客さんから予約申請が来る
管理画面の「予約申請」タブに通知が表示されます。

### ステップ2: 予約を承認
「承認」ボタンをクリック

### ステップ3: 施設予約サポートが表示される（施設予約が必要な場合）
```
┌─────────────────────────────────┐
│ 📞 施設予約が必要です             │
│                                   │
│ 亀田総合体育館                   │
│                                   │
│ [📞 電話をかける]  [🌐 Web予約ページ] │
│  025-382-3222     を開く          │
│                                   │
│ ☑️ 施設予約を完了しました         │
└─────────────────────────────────┘
```

### ステップ4: ボタンをクリック
- **電話予約:** 「電話をかける」をタップ → 自動で電話アプリが起動
- **Web予約:** 「Web予約ページを開く」をクリック → 予約ページが開く

### ステップ5: 施設予約が終わったらチェック
チェックボックスにチェックを入れる（予約漏れ防止）

### ステップ6: LINEメッセージをコピーしてお客さんに送信
「コピーする」ボタンで一発コピー

## 📝 設定方法（施設マスタ）

施設予約サポート機能を使うには、施設マスタに**電話番号またはURL**を登録する必要があります。

### 管理画面での設定
1. 管理画面を開く
2. 「施設管理」タブをクリック
3. 各施設の編集で以下を入力：
   - **電話番号**（例：025-382-3222）
   - **予約URL**（例：https://example.com/booking）
   - **予約方法**（電話 / Web / 両方）

**これで施設予約サポート機能が使えるようになります！**
