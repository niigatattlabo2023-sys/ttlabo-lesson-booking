## 📋 更新内容（最新版）

### ✅ タイトル変更
- **旧:** 新潟卓球ラボのレッスン予約
- **新:** 個人レッスン予約フォーム

すべてのページ（index.html、booking.html、admin.html）のタイトルとヘッダーを更新しました。

### ✨ 施設予約サポート機能の強化

管理画面で予約を承認した際、コーチが施設予約を担当する場合に以下の機能が追加されました：

#### 1. ワンクリックで施設に連絡
承認後の予約詳細画面に「施設予約サポート」セクションが表示されます：

```
┌─────────────────────────────────┐
│ 📞 施設予約が必要です             │
│                                   │
│ [📞 電話をかける]  [🌐 Web予約ページ] │
│  025-XXX-XXXX    を開く           │
│                                   │
│ ☑️ 施設予約を完了しました         │
└─────────────────────────────────┘
```

**機能詳細：**
- **電話予約の場合:** 
  - 電話番号をタップすると自動で電話アプリが起動
  - スマホなら即座に発信可能
  
- **Web予約の場合:**
  - ボタンをクリックすると施設の予約ページが別タブで開く
  - URLは施設マスタで設定可能

- **予約完了チェックボックス:**
  - 施設予約が完了したらチェック
  - 予約漏れ防止に役立つ

#### 2. 表示条件
施設予約サポートは以下の条件で表示されます：
- 予約ステータスが「承認済み」
- 施設予約担当が「コーチ」
- 施設情報（電話番号 or URL）が登録済み

#### 3. デザイン
- グラデーション背景（黄色×青）で目立つ
- 大きなアイコンとボタンで操作しやすい
- スマホでも使いやすいレスポンシブデザイン

## 🚀 セットアップ

### 1. Supabaseデータベース設定
1. https://supabase.com/dashboard にアクセス
2. 「SQL Editor」を開く
3. `supabase-schema.sql` の内容を実行

### 2. 管理者アカウント作成
1. Supabase Dashboard → 「Authentication」 → 「Users」
2. 「Add user」でメールアドレスとパスワードを設定

### 3. 施設情報の登録
管理画面の「施設管理」タブで施設を追加：
- 施設名
- 住所
- **電話番号**（施設予約サポート用）
- **予約URL**（施設予約サポート用）
- 予約方法（電話 / Web / 両方）

**重要:** 電話番号またはURLを登録することで、施設予約サポート機能が利用できます。

## 💡 使い方

### お客さんへの案内
予約フォームのURLをLINEで共有：

```
レッスン予約が簡単になりました！
以下のリンクから24時間いつでも予約できます🏓

📅 予約フォーム
https://your-url.com/booking.html

・施設と日時を選ぶだけ
・料金も自動計算されます
・確定次第LINEでご連絡します
```

### 予約が来たら
1. 管理画面を開く
2. 「予約申請」タブで新着確認
3. 「承認」ボタンをクリック
4. **施設予約が必要な場合:**
   - 表示された「電話をかける」or「Web予約ページを開く」ボタンをクリック
   - 施設予約完了後、チェックボックスにチェック
5. LINEメッセージをコピーして顧客に送信
6. Googleカレンダーに追加

## 📊 データベーススキーマ

### facilities（施設マスタ）
- `id`: UUID
- `name`: 施設名
- `address`: 住所
- `phone`: 電話番号（施設予約サポート用）
- `booking_url`: 予約URL（施設予約サポート用）
- `booking_method`: 予約方法（phone / web / both）

### lesson_requests（予約申請）
- `id`: UUID
- `customer_name`: 顧客名
- `customer_type`: 区分（junior / adult）
- `lesson_type`: レッスン種別（individual / group）
- `facility_id`: 施設ID
- `start_time`: 開始日時
- `duration_minutes`: レッスン時間（分）
- `participants`: 参加人数
- `facility_booking_by`: 施設予約担当（customer / coach）
- `coach_fee`: コーチ料
- `status`: ステータス（pending / approved / rejected / cancelled）
- `notes`: 備考

### blocked_times（ブロック時間）
- `id`: UUID
- `title`: タイトル
- `category`: カテゴリ（lab / family / practice / regular_lesson / other）
- `start_time`: 開始日時
- `end_time`: 終了日時
- `is_recurring`: 繰り返しフラグ
- `notes`: 備考
